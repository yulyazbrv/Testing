Тест-кейс 1: Проверка заполнения поля поиска
Шаги:
1. Зайти на сайт https://www.apple.com/
2. Нажать н6а иконку поиска в правом верхнем углу
3. Ввести данные для поиска
4. Нажать клавишу Enter либо на иконку поиска
Ожидаемый результат:
Вводимые значения												Результат						Оценка
<Поле заполненное только пробелами>						Перенаправления на страницу с найденной информацией – нет.																				Выполнено
<Поле заполненное информацией которой нет в каталоге>	Перенаправление на страницу с найденной информацией – есть, уведомление «Sorry no matches were found»									Выполнено
<Поле заполненное информацией которая есть в каталоге>	Перенаправление на страницу с найденной информацией – есть, вывод товаров удовлетворяющих введённой информации							Выполнено

Тест-кейс 2: Проверка авторизации в AppleId
Шаги:
1. Зайти на сайт https://www.apple.com/
2. Нажать на иконку Bag в правом верхнем углу.
3. Нажать на поле Sign in
4. Заполнить поле Apple Id
5. Нажать стрелку справа либо клавишу Enter
6. Заполнить поле Password
7. Нажать стрелку справа либо клавишу Enter
